# Programa Celso Furtado

Tema desenvolvido para o portal do Programa Celso Furtado em Ghost CMS.

O projeto tem como objetivo centralizar informações institucionais sobre o programa, incluindo sua base legal, histórico, formas de participação, equipe, eixos temáticos, publicações, materiais de referência, newsletter, espaço para escrita e Fórum Celso Furtado.

A proposta do tema prioriza uma experiência responsiva, acessível e organizada, alinhada à identidade visual do programa e à divulgação de conteúdos voltados ao desenvolvimento, cultura, educação e cidadania.

**Tecnologias utilizadas:**

- Ghost CMS
- Handlebars (HBS)
- HTML5
- CSS3
- JavaScript
